# 🚗 Vehicle Detection using YOLO

This project detects vehicles in images using YOLOv8 pretrained model.

---

## 🚀 Features

- Detect vehicles using YOLOv8
- Bounding box output
- Simple demo script

---

## 🛠 Tech Stack

- Python
- YOLO (Ultralytics)
- OpenCV

---

## 📁 Project Structure

```
vehicle-detection-yolo/
├── app.py
├── vehicle_detection.ipynb
├── requirements.txt
├── README.md
├── sample_output.png
```

---

## ⚙️ Run Project

```bash
pip install -r requirements.txt
python app.py
```
